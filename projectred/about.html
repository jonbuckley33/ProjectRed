Project Red is an all-javascript synchronous multiplayer game. It uses Box2DWeb to simulate all of the physics, has a NodeJs backend, and uses the CreateJs library for manipulating the HTML5 canvas. Additionally, all data is stored in JSON format. It's not meant to profit, or really demonstrate anything, but rather just to be a fun game for a couple friends. At least that's what it is to me.
