Projects

DataDrome Data Visualization

Worked for the summer at OSU's Department for Computer Science Engineering, making a web app to visualized geographically-tagged data interactively. This included support for the Microsoft Kinect and Leap motion sensor.

Soulfire Software Indie Game Dev

Two summers ago, I interned at an Indie Game Development Studio. I contributed primarily to two projects, "Fleshblender" , a 2D zombie game, and "Thieve's Plummet", an action arcade-style game. Fleshblender was developed for Windows8 using XAML UI design and a modified Box2D physics engine. Thieve's Plummet was developed in Unity 3D, and is targeting multiple platforms for deployment.

CMU Eats Open-Sourced Food Ratings

For HackCMU 2013, three friends and I developed a web app that hosted all of the dining option information for Carnegie Mellon. This included locations, hours information, and menus. In addition, users could vote up and down individual food items. This was developed with a PHP backend and MySQL data store.

Cube4 LLC Health Insurance Compliance Software Suite

Last summer, I interned at Cube4 LLC as a web developer. Cube4 provides software to employers for them to easily comply with the Affordable Care Act. My work spanned from things as low level as the domain model system to the user interface. The primary application is developed in .NET technologies and leverages the ASP.NET MVC 5 framework.

Project Red Javascript Action Platformer

While working for Cube4 last summer, some friends and I worked on a javascript-only action game. We accomplished this using the tools EaselJS and CreateJS. The game was served by a node js backend and can be accessed here .

Hanashi Anonymous Messaging Service

This fall, at MHacks 2014, my friend and I developed an ephemeral, anonymous messaging service. It was unique because the service focused the chats on the conversation rather than on the people, encouraging deeper conversation than the typical anonymous chat. This service was developed on top of Node JS and ran as a mobile-compliant web app.
