Projects

NOTE   Many of my personal projects' source code is publicly visible on github and can be viewed on my profile.

CMU Eats Crowd-Sourced Food Ratings

For HackCMU 2013 , three friends and I developed a web app that hosted all of the dining option information for Carnegie Mellon University. This included location data, open hours, and menu information. In addition, users could vote up and down individual food items. This was developed with a PHP backend and MySQL data store.

Project Red Javascript Action Platformer

While working for Cube4 one summer, some friends and I worked on a javascript-only action game. We accomplished this leveraging the tools EaselJS and CreateJS. The game was served by a Node JS backend and can be accessed here .

Hanashi Anonymous Messaging Service

At MHacks 2014, my friend and I developed an ephemeral, anonymous messaging service. It was unique in that the service focused chats on conversation rather than on the people, encouraging deeper conversation than the typical anonymous chat. This service was developed on top of Node JS and ran as a mobile-compliant web app.

Pong VR Google Cardboard Game

In the summer of 2016, I developed an interest in Virtual Reality. After attending a VR fair in Los Angeles (VRLA), I decided I would try my hand at some VR development. The result was a pong-esque game for the Google Cardboard that I developed in Unity. You can find the source code here.
